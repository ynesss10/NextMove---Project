# Next Move

## Project Description
Next Move is a web-based platform designed to help students explore their interests and skills in order to discover suitable career paths. The system provides career recommendations based on the interests and abilities selected by users.

---
## Installation
1. Clone the repository:
    ```bash
    https://github.com/ynesss10/NextMove---Project.git
    ```

2. Navigate into the project folder:
    ```bash
    cd Next-Move
    ```
 3. Import the database:
   - Open phpMyAdmin
   - Create or import the provided `database.sql` file

4. Run the local server:
    ```bash
    php -S localhost:3000 -t public
    ```

5. Open your browser and visit:
    ```plaintext
    http://localhost:3000
    ```

---

## Usage

1. Open the website in your browser.

2. Create an account through the Sign Up page.

3. Select your interests and skills from the available categories.

4. The system will provide career recommendations based on your selected interests and abilities.

5. Users can explore career details and save their preferred career choices.
